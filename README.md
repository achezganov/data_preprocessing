## Описание проекта
Этот проект представляет исследование развития игровой индустрии в начале XXI века (2000-2013 годы).

  

Проект включает:
- **Анализ игровых платформ** — изучение распространения основных консолей и систем
- **Исследование жанров** — сравнение объёмов продаж по различным категориям игр
- **Региональный анализ** — выявление предпочтений игроков в разных странах
- **Особый акцент на RPG** — детальное изучение компьютерных ролевых игр

## Данные проекта
Проект работает с историческими данными о продажах видеоигр из открытых источников.

  

### Источник данных
Файл: `/data/new_games.csv`

### Структура датасета

| Поле | Описание |
|------|----------|
|`Name`|Название игры|
| `Platform` | Платформа выпуска |
| `Year of Release` | Год выпуска |
| `Genre` | Жанр игры |
| `NA sales` | Продажи в Северной Америке (млн копий) |
| `EU sales` | Продажи в Европе (млн копий) |
| `JP sales` | Продажи в Японии (млн копий) |
| `Other sales` | Продажи в остальных странах (млн копий) |
| `Critic Score` | Оценка критиков (0-100) |
| `User Score` | Оценка пользователей (0-10) |
| `Rating` | Рейтинг ESRB |

## Основные задачи
### 1. Загрузка и исследование данных

- Загрузка датасета и необходимых библиотек Python
- Изучение структуры и размера данных
- Анализ типов данных и наличия пропусков
- Проверка соответствия данных описанию

### 2. Проверка и предобработка данных
- **Нормализация названий столбцов** — преобразование в snake_case
- **Корректировка типов данных** — приведение к надлежащим форматам
- **Обработка пропусков** — заполнение или удаление недостающих значений
- **Устранение дубликатов** — поиск и удаление явных и неявных дубликатов
- **Нормализация текстовых данных** — унификация написания названий и жанров

### 3. Фильтрация данных
Отбор игр за период **2000-2013 года включительно** для анализа истории развития индустрии в начале XXI века.

### 4. Категоризация оценок
#### Оценки пользователей (User Score)
- **Высокая оценка** — 8.0–10.0 (включая границы)
- **Средняя оценка** — 3.0–7.9
- **Низкая оценка** — 0.0–2.9

#### Оценки критиков (Critic Score)
- **Высокая оценка** — 80–100 (включая границы)
- **Средняя оценка** — 30–79
- **Низкая оценка** — 0–29

### 5. Анализ платформ
Выделение **топ-7 платформ** по количеству выпущенных игр в период 2000-2013 годов.

## Дополнительная информация
### О рейтинге ESRB
Система рейтинга **ESRB** (Entertainment Software Rating Board) определяет возрастные категории:
- **RP** — Rating Pending (рейтинг ожидается)
- **EC** — Early Childhood (от 3 лет)
- **E** — Everyone (для всех)
- **E10+** — Everyone 10+ (от 10 лет)
- **T** — Teen (от 13 лет)
- **M** — Mature (от 17 лет)
- **AO** — Adults Only (от 18 лет)
